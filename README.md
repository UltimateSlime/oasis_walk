# oasis_walk

A 3D exploration game built with [Bevy](https://bevyengine.org/) 0.18 and [Avian3D](https://github.com/Jondolf/avian) physics.

## Controls

| Key | Action |
|-----|--------|
| WASD | Move |
| Mouse | Look around |
| Shift | Run |
| Space | Jump / Double-jump to float |
| Ctrl | Crouch |
| F | Toggle fly mode |
| V | Toggle TPS / FPS camera |
| Esc | Quit |

## Build

```sh
cargo build
cargo run
```
